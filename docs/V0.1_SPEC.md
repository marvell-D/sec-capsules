# v0.1 Spec

v0.1 is a vertical slice for the `sec-capsules` runtime.

## Required Capabilities

- Load built-in capsule cards.
- Generate safe command plans.
- Enforce scope before real execution or fixture replay.
- Preserve raw output as artifacts.
- Parse raw output into structured security objects.
- Emit token-budgeted `ObservationPacket` objects.
- Expose a CLI and a thin MCP adapter.

## Built-In Capsules

- `httpx`
- `katana`
- `nuclei`

## Runtime Boundary

The runtime does not plan attacks. It only handles tool invocation lifecycle tasks.
